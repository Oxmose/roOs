void kickstart ( void  )

Main boot sequence, C kernel entry point.

Main boot sequence, C kernel entry point. Initializes each basic drivers for the kernel, then init the scheduler and start the system.

Warning
This function should never return. In case of return, the kernel should be able to catch the return as an error.
