OS_RETURN_E pit_set_handler ( void(*)(kernel_thread_t *)  handler)

Sets the PIT tick handler.

Sets the PIT tick handler. This function will be called at each PIT tick received.

Parameters
[in]handlerThe handler of the PIT interrupt.
Returns
The success state or the error code.
  • OS_NO_ERR is returned if no error is encountered.
  • OS_ERR_NULL_POINTER is returned if the handler is NULL.
    • OR_ERR_UNAUTHORIZED_INTERRUPT_LINE is returned if the PIT interrupt line is not allowed.
  • OS_ERR_NULL_POINTER is returned if the pointer to the handler is NULL.
  • OS_ERR_INTERRUPT_ALREADY_REGISTERED is returned if a handler is already registered for the PIT.
