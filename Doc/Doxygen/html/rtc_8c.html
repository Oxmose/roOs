void rtc_update_time ( void  )

Updates the system's time and date.

Updates the system's time and date. This function also reads the CMOS registers. By doing that, the RTC registers are cleaned and the RTC able to interrupt the CPU again.

Warning
You MUST call that function in every RTC handler or the RTC will never raise interrupt again.
