Assert macro used by the scheduler to ensure correctness of execution. Due to the critical nature of the scheduler, any error generates a kernel panic.

Parameters
[in]CONDThe condition that should be true.
[in]MSGThe message to display in case of kernel panic.
[in]ERRORThe error code to use in case of kernel panic.
