INTERRUPT_TYPE_E(* driver_handle_spurious) (const uint32_t int_number)

The function should check if the serviced interrupt is a spurious interrupt. It also should handle the spurious interrupt.

The function should check if the serviced interrupt is a spurious interrupt. It also should handle the spurious interrupt.

Parameters
[in]int_numberThe interrupt number of the interrupt to test.
Returns
The function will return the interrupt type.
  • INTERRUPT_TYPE_SPURIOUS if the current interrupt is a spurious one.
  • INTERRUPT_TYPE_REGULAR if the current interrupt is a regular one.
