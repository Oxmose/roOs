\doxysubsection*{UTK -\/ Utility Kernel}


\begin{DoxyItemize}
\item UTK is a kernel created for training and educational purposes. It is designed to execute in kernel mode only.
\item UTK has a configuration file allowing the kernel to be customized depending on the system it will run on.
\end{DoxyItemize}

\doxysubsubsection*{Memory Management}


\begin{DoxyItemize}
\item Higher-\/half kernel
\item Virtual/physical memory allocator
\item Process isolation
\end{DoxyItemize}

\doxysubsubsection*{Synchronization}


\begin{DoxyItemize}
\item Futex based synchronization
\item Mutex\+: Non recursive/\+Recursive -\/ Priority inheritance capable. Futex based
\item Semaphore\+: FIFO based, priority of the locking thread is not relevant to select the next thread to unlock. Futex based
\item Spinlocks
\item Kernel critical section by disabling interrupts
\end{DoxyItemize}

\doxysubsubsection*{Scheduler}


\begin{DoxyItemize}
\item Priority based scheduler (threads can change their priority at execution time)
\item Round Robin for all the threads having the same priority
\end{DoxyItemize}

\doxysubsubsection*{Process / Thread management}


\begin{DoxyItemize}
\item Fork / Wait\+PID
\item Kernel threads support
\item System calls
\end{DoxyItemize}

\doxysubsubsection*{BSP Support\+: i386/x86\+\_\+64}


\begin{DoxyItemize}
\item PIC and IO-\/\+APIC support (PIC supported but IO APIC required)
\item Local APIC support with timers
\item PIT support (can be used as an auxiliary timer source)
\item RTC support
\item Basic ACPI support (simple parsing used to enable multicore features)
\item Serial output support
\item Interrupt API (handlers can be set by the user)
\item 80x25 16colors VGA support
\item Time management API.
\end{DoxyItemize}

 \doxysubsection*{Build and run}

To build UTK, choose the architecture you want and execute.

Architecture list to use in the TARGET flag\+:
\begin{DoxyItemize}
\item x86\+\_\+i386
\item x86\+\_\+64
\end{DoxyItemize}

\doxysubsubsection*{Compilation}

make target=\mbox{[}TARGET\mbox{]} TESTS=\mbox{[}TRUE/\+FALSE\mbox{]} DEBUG=\mbox{[}TRUE/\+FALSE\mbox{]} TRACE=\mbox{[}TRUE/\+FALSE\mbox{]}

\doxysubsubsection*{Execution}

make target=\mbox{[}TARGET\mbox{]} run

\doxysubsubsection*{Tests and Debug}


\begin{DoxyItemize}
\item TESTS flag set to TRUE to enable internal testing
\item DEBUG flag set to TRUE to enable debuging support (-\/O0 -\/g3)
\item TRACE flag set to TRUE to enable kernel tracing 
\end{DoxyItemize}
